# Инструкция по публикации на GitHub

Для публикации проекта "Супер Кликер" на GitHub выполните следующие шаги:

## 1. Создайте токен доступа к GitHub

1. Войдите в свой аккаунт GitHub
2. Перейдите в Settings → Developer settings → Personal access tokens → Tokens (classic)
3. Нажмите "Generate new token"
4. Выберите scopes: `repo`, `public_repo`
5. Скопируйте токен и сохраните его

## 2. Запустите скрипт публикации

1. Запустите файл `create_github_repo.bat`
2. Введите ваш GitHub токен при запросе
3. Введите имя репозитория (по умолчанию: super-clicker)
4. Скрипт автоматически создаст репозиторий и загрузит файлы

## 3. Альтернативно - через веб-интерфейс

Если скрипт не работает, вы можете загрузить файлы вручную:

1. Создайте репозиторий на GitHub через веб-интерфейс
2. Загрузите все файлы из папки `C:\Users\Дмитрий\Desktop\кликер\`:
   - `index.html` - основная игра
   - `telegram_bot.py` - Telegram бот
   - `web_app.py` - веб-приложение
   - `requirements.txt` - зависимости
   - `README.md` - описание
   - `FULL_SETUP.md` - инструкция
   - `.gitignore` - исключения для git
   - `start_ngrok.py` - скрипт для ngrok
